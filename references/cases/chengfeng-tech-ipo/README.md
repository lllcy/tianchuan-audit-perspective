# 成都乘风流体 IPO 案例包

这是一个案例型资料目录，不属于 skill 的通用方法论主干，而是用于补充展示“田川式审计视角”在具体 IPO 场景下的落地分析方式。

## 文件说明

- 01-sponsor-report.md：券商/保荐机构视角，偏行业、板块匹配和申报逻辑
- 02-expert-report.md：审核专家视角，偏风险排序、问询预判和审核关注点
- 03-fraud-specialist-report.md：舞弊专员视角，偏异常模式、动机和验证框架
- 04-compilation-report.md：四角色合编结论，适合作为案例总览入口

## 建议用法

如果只是维护这个 skill 的通用能力，优先看 references/research 下的四份主研究文件。

如果要补充案例、训练问询回复思路、或沉淀某个具体项目的多角色判断，再看这个案例目录。

## 归档原则

后续新增案例建议继续按下面的结构放：

```text
references/cases/
  [case-name]/
    README.md
    01-sponsor-report.md
    02-expert-report.md
    03-fraud-specialist-report.md
    04-compilation-report.md
```

这样通用方法和个案材料不会混在一起。
