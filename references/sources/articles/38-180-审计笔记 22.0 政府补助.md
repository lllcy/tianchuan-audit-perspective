# 审计笔记 22.0 政府补助

前言

政府补助准则本身不是非常复杂，但并不排除实务上会碰到坑的情况...首先，用最直白的话总结：1、从政府取得；2、无偿性....16号准则啥都可以记不住，但是这两个点是必须记住的...要不干活的时候指不定就翻车了...

对于政府补助的判断，不要一上来就判断是资产相关还是收益相关，如果“补助”不适用于16号准则...那自然也就不会有后面的资产相关还是收益相关的判断了...

（一）政府补助的判断

就如前言所说的，政府补助相关科目需要优先判断是账载的“政府补助”是否真正属于政府补助相关内容，正着不好解释，咱们就反着来...

1.从政府取得

就比如某国家级重点课题，A公司为课题唯一参与方（假定满足政府补助准则相关要求），那么A公司以项目参与方为由向其子公司B支付的款项，B对于收悉款项是否可按照政府补助进行会计处理那？

以下为个人理解，如有不妥之处，还望指正....因为A公司是课题唯一参与方，即使B公司作为“参与方”实际参与了A公司的课题研究中，但对于B而言，首先A不是政府部门；其次，B公司作为“参与方”如果需要向A公司提供人员或技术支持，更接近于完成一项履约义务，即适用收入准则...

“如果B啥也不用干那？”

“那他凭啥拿补助....”

如果出现上述这种躺着拿"补助"的情况，因为实际不具有商业实质，更接近于一笔资金划拨（其他往来款）....PS：这种情况很多时候是为了拉一把子企业的经营业绩，或“税收筹划”考虑....但明眼人一看就知道是在瞎折腾...

2.无偿性

无偿性其实有时候会有些模糊，就比如“简称年产XXX吨的生产线”、“形成一项国家领先技术”、“年纳税XX万元以上"....这种约定其实更像是一种“兜底”性质的约定，因为政府（除了国家级政策外）通常不会不附加任何条件就拨付大额的政府补助款项，所以会做一些底线条件的约定，但这些约定往往并不等同于“有偿”。

还是反例会说的比较清楚...如果文件中约定“产权归属政府”、“未经授权、审批获准，不得对外使用”、“XXX前交付XXX产品/项目/软件”，诸如此类的约定往往都明示或暗示相关约定不符合“无偿性”，即不适用于政府补助准则。

上面举了两个（可能不太靠谱）的例子，但对于政府补助项目的作业，务必记好“从政府取得”和“无偿性”两个重点中的重点...对于不满足这两个条件中的任何一条，都不适用于政府补助准则，可能适用适用收入准则或其他准则，这就要具体问题具体分析了。

（二）政府补助的分类

先说实务上的重点...

1.不可能没有补助文件：除了国家部委发布的普惠性文件以外，无论是省级、市级还是区级的补助，都一定有补助文件，只是文件的形式会不太一样，会存在大政策附名单的类型，也可能是针对企业出具的单独文书，但文件一定是有的...为什么非要提这个点，是因为有时候被审计单位会说“钱都到账了，我这没收到文件”等事由，不提供文件...没了文件，自然也就不具备判断会计处理是否正确的基础。

提这个点，并不是让大家去干仗...文件的形式有很多种，且政府补助的申请有时候并不通过财务部门（可能由办公室或技术部门牵头申请），所以就导致原始凭证后附资料里可能存在文件缺失，这种时候一是可以通过具体申请部门了解下文件情况，另一方面可以通过网上查询相关项目的公示信息。就个人理解，无论是线上资料还是线下的项目申请都是政府补助相关科目不可或缺的审计证据...

当然，上述的前提是补助金额重大...如果真的是那种一年发个千八百块的补助...是，只要是政府发的钱，就一定有文件出处，但是基于重要性的考虑，切不要走火入魔...

2.基于文件的职业判断：如果能够顺利取得文件，文件本身也有可能约定的比较模糊，这个时候就需要查阅政府补助项目的申请文件（如果补助文件内容清晰，此项内容为补充支持内容），目的很简单，就是看补助或申请补助的事由是：①与资产相关还是与收益相关；②是已经发生的投入还是将会发生的投入。

对于①而言，查阅的重点主要是为判断政府补助项目属于资产相关还是收益相关，通常情况下资产相关的表述包括“为构建XXX产线，预计投入XXX万元，其中:企业自筹80%，申请补助20%”、“XXX年已建成XX产线并投入批量生产，满足XXX号文件适用申请指标”；收益相关的表述包括“为形成XXX技术，预计投入人工成本XXX小时，约XXX万元”、“XXX年，因XXX业务合计发生支出XXX万元...满足XXX号文件适用标准”；

对于②而言，了解相关投入/支出是否发生，主要是因为“先投入、后补助”和“先补助、后投入”的会计处理有一些细微的差别，具体表现为：

资产相关的补助：如果企业先取得与资产相关的政府补助，再确认所购建的长期资产，总额法下应当在开始对相关资产计提折旧或进行摊销时按照合理、系统的方法将递延收益分期计入当期收益；如果相关长期资产投入使用后企业再取得与资产相关的政府补助，总额法下应当在相关资产的剩余使用寿命内按照合理、系统的方法将递延收益分期计入当期收益。

收益相关的补助：用于补偿企业以后期间的相关成本费用或损失的，确认为递延收益，并在确认相关成本费用或损失的期间，计入当期损益或冲减相关成本；用于补偿企业已发生的相关成本费用或损失的，直接计入当期损益或冲减相关成本。

需要补充说明的是，如果一项补助同时包含与资产相关和与收益相关的政府补助，需要拆分对应的补助金额并相应进行会计处理，切勿一锅端...

注：以上处理均为总额法模式下，净额法的适用方法可查阅16号准则及其应用指南。

3.会计估计的判断：估计这个事，既然叫估计实际上就是没有准确答案的意思，但也不是说能拍大腿去乱估计...对于政府补助，特别是资产相关的政府补助，要尽可能保证摊销的方式与资产的口径平行，当然，实务中会出现补助的资产标的的折旧年限不一致的情形，如同时对购置土地及构建厂房进行补助，则需要考虑按照适用的比例（如补助占预计投入的比例乘以对应资产的投入分摊），具体的操作请征求项目负责人意见...以上内容仅供参考...此外，收益相关的政府补助也不是全都是一次性进损益的，比如针对特殊贷款的贴息补助，也要考虑贷款的期限，而不能简单的在收悉补助当期一把全塞进损益里...

（一）审计说明

这里就不再分主表明细表去一张纸的写了，要不又会罗里吧嗦的一大堆...

审计说明的目的是为了让查阅底稿的人可以清晰地了解作业人员的作业思路和作业结果，所以请不要就写一句“主表明细表一致，未见异常”...求求了...

个人建议的审计说明举例如下：

“1.截止至本期期末，被审计单位递延收益结余金额XXX万元，较期初结余增加XXX万元，增幅XX%。XXX年X月，XXX部门就XXX事项下发补助文件，对辖区内满足XXX条件的法人主体进行补助。被审计单位于XX年XX月提交申请并获批取得关于XXX项目的补助合计金额XXX万元；

2.我们已取得前述补助文件（索引号：XXXX），并通过公开信息查阅了相关公示信息（网址：https://...）（如适用），针对补助文件的检查过程详见<递延收益/其他收益/营业外收入-补助文件检查>底稿；

3.我们取得并查阅了被审计单位新增政府补助的银行进账单，检查了补助单位与付款单位是否一致、补助金额是否与文件金额一致，检查过程未见异常，底稿详见<递延收益/其他收益/营业外收入-抽凭>底稿；经上述程序检查，本年大额新增补助金额确认准确，未发现重大异常。

4.除上述事项外，针对以前年度取得的大额政府补助，我们取得了被审计单位编制的摊销测算表，了解了本年大额政府补助项目的实施情况，取得并检查了本期验收并开始进行摊销项目的确认依据，底稿详见<递延收益/其他收益/营业外收入-新增检查>及<递延收益/其他收益/营业外收入-摊销测算>底稿；

5.经审计，我们关注到XXX项目政府补助摊销期限未按照资产剩余年限摊销，我们就差异金额与被审计单位管理层进行了沟通，管理层认可了我们的测算结果，并进行如下调整（列示调整分录）/管理层认为差异较小，我们对未更正错报进行了汇总评估，详见<未更正错报汇总评估底稿>，累计未更正错报低于实际执行的重要性水平，我们认为相关错报金额可接受。”
