# 审计笔记8.1 存货（原材料）

![image.png](https://www.cgzcpa.com/uploads/image_c4cb9eb5e4.png)

**2-1.****原材料**

原材料是存货核算的起点，也是制造型企业生产的主要成本构成项目，当然视具体产品而言占比波动还是比较大的。当然除了制造型企业，能源型企业、药品行业、快消品行业等都会涉及原材料的核算，说白了原材料就是做菜用的料...干炒空气是做不出一盘菜的...

**2-1-1** **收发存报表**

**如引言部分所述，下面来详细讲一下收发存的分析方法，大写加粗的重点！**习惯性上企业一般会提供全年所有原材料的结存及收发情况，比较典型的表格大概是这个样子：**存货示例表格****\-1**

![image.png](https://www.cgzcpa.com/uploads/image_7a0a367d48.png)

**首先，来逐项说明一下：**

-   物料编码及物料名称通常情况下是唯一的，为什么？因为不唯一的话谁知道领得是花生米还是葱花...在盘点、登记入库及领料出库时，物料编码及名称是库管员的唯一索引，依靠唯一编码与采购入库的物料名称进行对应并办理登记入库；根据生产下达需求所记录的具体物料办理领料登记及领料单；在实际盘点时，根据物料卡片上注明的编码及型号与财务系统中记录的数量进行核对。
-   期初数量及期初金额表示资产负债表日期初账载明细物料的数量及金额，多嘴说一句，加个和看看和期初的余额表能不能对上...因为企业导的收发存很可能是11月至12月的，而非1至12月的....拿单月的做程序就坐等拉胯吧。
-   入库数量为库存管理员根据采购物料编码登记办理入库数量，比如买了两头葱，1斤花生米，需要说明的时在了解内部控制活动的时候，建议问询了解入库单据的传递方式，是通过系统扫描（扫码枪）自动登记、仓库管理员系统内登记还是填写纸质单据传递给财务，再由财务登记入账，三者的区别在于自动化程度逐渐降低，通常情况下人工维护的占比越高，公司自动化核算的水平越低，因为人为因素导致的单据传递不及时、可能产生的未准确暂估的错报可能性就越高。当然，也要考虑原材料的周转频率，比如做雕塑的...一年就买一块大理石，就算刻块碑当入库单时间也够用...所以并不是说人工的就一定会有错报，自动化就一定没有错报，还是要结合具体业务情况具体分析。
-   入库金额较入库数量会相对复杂一些，why？因为库管只管数量不管金额，财务如果不来发票大概率也不知道对应材料的金额，所以就会涉及采购部门反馈对应批次材料的预计采购价格....这又会类比与入库数量，如果在下达采购计划时，对应物料的协议约定金额以登记进系统，在库管扫描或登记对应批次材料时，系统可自动对应单价，那财务直接就解放了，只要等发票来了消暂估就好了....好矮窝..很多企业是实现不了的，所以在无法自动生成入库单价时，原材料暂估金额可能会存在重大错报风险。注意这里是准确性的认定，不同与未及时办理产生的截止认定错报风险，如果和审计书写的不一样，请以审计教材为准...\_(:з」∠)\_。
-   出库金额类似于入库数量，是仓库管理员根据生产计划下达的时间及物料需求，向对应产线发出对应材料的记录过程，即原材料自原材料仓库发至产线，由存货\-原材料科目结转至生产成本\-直接材料科目。
-   接化发，哦不对...收发存的难点来了...就是出库金额的问题，运气好的月末加权平均大家相安无事.....万一赶上SAP的移动加权平均....或者是风骚的批次法.....那只能祝你幸福了....月末加权平均如果考过会计的应该都会算，木有考过会计的请去看会计书....简单的地说，月末加权平均就是全月放弃治疗，月底的时候根据月初结余和本月累计发生额一次性补救，**下面的公式大写加粗的重要****.**...：

   ***当月出库单价****\=****（月初结存金额****+****本月入库金额）****/****（月初结存数量****+****本月入库数量）**    **当月出库金额****\=****当月出库单价** **X** **当月出库数***所以，在这种计算方式下，入库数量、金额及出库数量只要有一个整错了就必然拉胯....其次...没有其次了....什么？结存还没有说？结存就是期初+入库\-出库....（完）....  
  
 

**2-1-2** **常用的分析性程序及后续实质性程序**

     **A** **入库分析**首先，有没有其次等写到了再说....收发存报表基本是开底稿时唯一可以取得的数据资料，毕竟不能一上来就撸入库单、出库单吧类似于费用类底稿的两期对比分析，原材料包括产成品也可以进行类似的分析，比如上面的表格中鸡腿肉的结存单价从期初的20元（金额/数量）上涨到了期末的24.95元，当然原因其实是显而易见的，是由于本期入库的均价25元（入库金额/入库数量）明显高于了期初结存单价，那么分析就完了嘛？必然不是...这就好比用1+1=2去解释为什么2=1+1...这也就是为什么说类似于费用底稿，在已知发生额存在重大波动的情况下，需要细化分析的维度，由年化维度扩展至月化维度，沿用上表数据举例如下：**存货示例表格****\-2**

![image.png](https://www.cgzcpa.com/uploads/image_866bf21a58.png)

如果单纯的看全年的入库单价，其实很难直观的发现波动的原因，但是如果按照月度打开就不难发现2月的采购单价明显异常，所以细节测试也好，计价测试也好，包括抽凭选几月更合适那？分析性程序原则上来说颗粒度越细反而越好，因为便于审计人员快速的识别异常，并有针对性的执行审计程序，而不是跟没头苍蝇似的抓起本凭证来就开始登记入库单号....围绕着上面的情况，展开点说：在入库单价存在异常波动的情况下并不急着下出现错报的结论，而是先要了解业务的重大变动，比如2月份鸡都变异只长了一条腿...禽流感、熔岩布、10纳米芯片....这里还要额外说一句，特殊行业的原材料会存在非常剧烈的波动，比如化工行业，所以在做存货前还是需要具备一定的行业知识，否则绝对会满脑子黑人问号的.....出现异常单价的简易判断逻辑：

![image.png](https://www.cgzcpa.com/uploads/image_82aed2c039.png)

单价的异常波动，特别是临近资产负债表日的激增与激减，包括存在重大关联采购、采购量与实际产能明显不匹配等，都表明可能存在舞弊风险，在应对时应考虑扩大细节测试样本量及审计程序覆盖期间。必要时应考虑结合应付账款及预付账款科目变动，检查供应商付款流水、物流发运记录等信息，判断业务是否真实发生。

     **C** **结存分析**结存分析有时候使用效果要优于出入库分析，本质是因为结存的数量与金额为时点状态，是相对静态的数据，而且可以结合盘点程序进行复核。且记，不要用分析性程序去替代细节测试等实质性程序，分析是为了便于审计作业人员快速识别风险与异常，有针对性的设计审计程序，而不是通过分析性程序形成审计结论。具备经验的情况下有效的执行分析程序有助于提高风险的应对效果，但切不可完全依赖分析程序，仅依靠主管判断形成结论，缺失必要的审计证据。结存分析可以从两个维度入手，一个是数量维度，一个是金额维度。数量维度又可以细分为数量增加与数量减少，一般情况下个人习惯用30%为标准去界定绝对变动值（计算波动时可以考虑套用ABS函数）。（1）当原材料期末结存数量较期初明显增加时，可以通过访谈了解原材料备料增加的原因，相关材料的采购、结余是否与生产预算或在手订单相匹配，如存在无订单支撑的预采购原材料，可能需要判断公司采购、生产循环是否存在内控缺陷。可能存在的错报风险详见**2-7** **减值测试。**（2）当原材料期末结存数量较期初数量明显减少时，应考虑该种原材料为核心原材料还是零星使用的材料（结合发生额判断），如为零星使用的材料那么整体变动对公司主要产品构成实质性影响，但如果为报告期内主要使用的材料那么需要了解结余减少的原因，是否存在滞销原因导致的减产，进而识别可能存在的原材料及产成品减值风险。  
  
 

**2-1-3** **原材料常见风险事项**

-   暂估不及时，实际已耗用未及时暂估；
-   入库、领料单据传递不及时，财务数据滞后；
-   入库、领料单据遗失、编号不连续，需考虑属于偶发还是广泛存在；
-   料号登记错误，导致出入库串料，个别材料出入单价波动异常；
-   只采不领、不采不领的重大结余材料

    来看一个经典案例....（以下节选自“证监会会计部：证券审计与评估行政处罚案例解析（2019）”）

万福生科通过虚构销售、自有资金循环、虚构生产和采购等进行“一条龙”式的财务造假。万福生科每月先确定虚假销售金额及对应的客户名单，并按照已确定的虚假销售金额编制虚假销售明细（包括日期、客户、产品类别、数量、单价和收入金额），开具发票；随后，根据虚假销售明细和固定成本参数计算应采购的原材料数量和金额，并编制从采购到领料环节等整套虚假单据（包括采购合同、入库单、领料单、公司开具的农产品收购发票等）。对于资金流转,公司先将资金转入其控制的个人银行账户,确认为采购付款;随后,将自有资金通过其控制的个人账户转回公司,确认为销售回款。公司按照成本参数确定投入产出比例,将商品销售与生产进行挂钩,确认生产成本;最后,以支付在建工程、预付工程款等名义将虚增的货币资金转移出公司,“消化”虚增的利润。经查,万福生科通过集采购、生产、销售于一体的“一条龙”式财务造假,于2008-2010年分别虚增销售收人12,262万元、14,966万元、19,074万元,虚增营业利润2,851万元、3,857万元、4,590万元。扣除上述虚增营业利润后,万福生科2008~2010年扣除非经常性损益的净利润分别为\-332万元71万元、383万元,不符合公开发行股票的条件。

一条龙的方式其实是最难识别的，因为涉及“精心准备”的舞弊，但整个过程中也不缺乏识别风险的途径：

从行业看，万福生科所处的稻米加工行业早在2009年就开始发生重大变化，全国大米加工企业因为原材料供应不足爆发了大面积的停产危机，开工率仅为30%~40%,但在万福生科的招股说明书中，本地原料供应充足，2009年收入较2008年增长达44%，米业毛利率丝毫没有受到原材料供应不足的影响。此外，在原料价格上涨和生产成本急剧上升，行业毛利率呈下跌态势的情况下，万福生科的销售毛利率下跌却并不明显。

从公司自身看，万福生科的业务特点和交易模式为管理层设计一条龙”式造假提供了良好的机会，一是稻米精深加工生产出的大米淀粉糖、大米蛋白属于小众产品，稻米精深加工行业规模小、可对比同业公司少、行业数据匮乏；二是主要原材料的大米大多数是通过当地个体工商户或自然人采购，而大米淀粉糖、大米蛋白等产品也主要是通过当地的中小客户来销售,这些供应商、客户大多属于缺乏有效内部控制的个体工商户或“夫妻店”式的小型企业,万福生科与其串通的风险较高；三是公司既可以代农户开具采购发票,也可以开具销售发票，佐以生产环节的内部单据,容易虚构完整的票据流；四是公司控制了若千供应商个人的银行账户，容易虚构完整的资金流。在本案中，审计师未能结合公司业务特点和交易模式等方面进一步识别和评估相关舞弊风险。

**2-1-4** **原材料小结**

原材料是一般制造型企业生产的起点，是整个存货科目的地基，在计划阶段建议：（1）访谈被审计单位核心生产人员、销售人员，旨在了解公司上下游情况，辨析是否存在可比上市公司，通过招股说明书、年报等公开信息初步了解行业的基本情况；（2）了解主要原材料供应商、供货周期、安全库存量，便于结合收发存、原材料明细表等财务资料进行分析性检查；（3）了解主要产品（属于提前预热）生产周期、料耗占比，便于根据实际账务处理结果复核是否与生产情况存在重大差异，如料耗占比明显偏高/低、生产周期过长/短等。
