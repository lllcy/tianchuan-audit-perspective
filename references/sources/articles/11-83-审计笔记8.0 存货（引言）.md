# 审计笔记8.0 存货（引言）

**一、引言**

存货属实是太庞大了，对于制造型企业存货的复杂程度不亚于其他报表科目的综合...如果说其他科目是一个壮汉，存货约等于少林寺十八铜人...

**（一）存货的难具体体现在什么地方？**

      首先，如果只是单纯的履行审计程序，存货并不像想象中的那么复杂，只要按（一万）部就班就好。主要的程序包括出入库单价核对、暂估分析、计价测试、料工费分配、成本倒扎、截止测试、存货盘点、减值测试、成本结转数量匹配、毛利分析....（好像也挺复杂的..）跑题了，其实上面说的（可能也许大概..）并不难，真正难的是搞清存货的核算流程、成本的分配方式，熟悉主要产品的加工工艺、加工流程、成本波动的业务原因，当然如果只是单纯的作为一名萌新，这样的要求多少有些过分，但在不充分了解上述事项的情况下，很难说把存货做通了。

      举几个简单的例子吧，比如一个做电视的公司，电视的标准组件有哪些？屏幕总得有吧....假设没有在产品的前提下，领了八块屏幕做了十台电视...这多少有点不科学，毕竟不能把屏幕锯成两段吧...又比如生物制药企业，在生产过程中会面临试剂失活等情况，100%成品率的试剂基本是不存在的...其实很多的风险事项并不能单纯的从财务报表或企业提供的辅助核算中体现出来，如果搞懂搞透行业的特性、产品的特性，执行少量的细节测试反而能更好的发现真正的风险。

      其次，存货的核算因企业而异、因行业而异....因会计而异，很多时候按固有模式去套是不可行的，毕竟成本要推倒重算是要出人命的...所以在很多时候是对核算的优化，而不是按固有认知全面否定，这就需要有一定程度的经营。比如IPO项目，在申报时需要列示产品的成本构成、料工费权重、各期产能与资产投入及人员投入的配比关系等，如果在进场时发现被审计单位未设置相关的辅助核算，那么建立有效的辅助核算体系比单纯的发现成本存在跨期要更有实际意义，因为单独的错报可能仅影响一个时点的财务数据，但系统性的缺陷可能直接影响一个项目最终的申报结果。所以存货底稿并不急着开，在经验不充足的情况下多看看同行业的招股书或反馈意见在很大程度上能提高对项目的认知，包括在建议客户完善核算的沟通时，也会有据可依。

      当然，最困难的点在于存货核算方式有问题的时候，客户一脸懵逼的看着审计旺，审计旺也一脸懵逼的看着客户....都知道不对，但都不知道咋改的时候最尴尬...

**（二）要如何下手？**

      存货核算的难点主要在于解决料工费的分配问题（约等于没说...）

     **1-1.****材料**

      材料一般分为主要材料和辅助材料，比如做一盘宫保鸡丁，鸡腿肉和花生米是主要材料，至于葱段和花椒放多少看厨子喜好了....比如做一双皮鞋，耗用的皮革基本是固定，但使用的胶水、缝合线可能会存在细微的差异，很难做到将胶水的用量定制化到每一双鞋上，这个时候就会涉及到耗用胶水在每一双皮鞋上分配多少成本的问题。

      这个时候就会涉及到一个叫**BOM**的东西，即物料清单，大白话就是配方...根据工艺需求，在加工前一般会设计生产方案、生产图纸，对生产产品需要耗用的直接及间接材料进行规范，比如做一部手机需要的芯片规格、屏幕尺寸等，其最核心的意义是在于制定该种产品的**材料定额**，材料定额制定的偏差率直接影响产品的成品率进而影响产品的实际毛利。沿用上面皮鞋的例子，一双鞋本来需要10块钱的皮革、1块钱的胶水，一般的产品报价方式是使用成本加成模式，在设计11元的成本前提下对外报价，比如22元，计划毛利率50%，但是因为设计偏差导致耗用了20块钱的皮革和4.2块钱的胶水，按原报价，实际毛利为\-10%。

      毛利的高低很大程度上受材料采购成本、人工水平及制造费用的影响，但是成品率及实际耗用差异这类隐形的损耗往往更为致命，因为采购价格是可以谈的...但是活儿干的槽是真的没救...

      所以，在做尽职调查也好、IPO进场阶段也好，选取主要产品的BOM及实际耗用情况进行比对，可以在一定程度上了解被审计单位产品加工工艺的成熟度及单位成本中材料耗用情况的稳定性，有助于后续进行毛利分析等工作。同时，因为物料清单提供了产品生产所需要耗用的主要原材料及辅助材料，对检查产成品材料耗用归集完整性上亦具有重大指导作用，比如宫保鸡丁没放花生米...这个成本多少有点不完整...

      **1-2.****人工**

      对应着材料定额，其实产品也是有人工定额的，即生产一双皮鞋、炒一盘菜需要的标准时间是多少，毕竟不能按佛跳墙的时间去炒宫保鸡丁吧...简单的来说比如做宫保鸡丁需要5分钟、糖醋里脊6分钟，假设各炒了10盘，那么合计的人工定额工时为110分钟，假设当天厨师的工资是550元，那么划分给宫保鸡丁的人工成本为250元，糖醋里脊300元....当然，这是理想情况下，实际执行的时候耗时可能为100分钟也可能为120分钟，就会涉及到人工的下一个问题，即实际工时与计划工时间如何分配的问题。是完全按照实际工时分割人工成本还是按照工时定额进行分配，就取决于产品的状态和复杂程度，如果是高度成熟流水线作业，实际工时与计划工时偏差率极低，那么使用计划工时可以节省统计实际工时的时间；如果手工程度高，产品工时波动很大，那么计划工时反而不能客观反映实际的人工成本。

    **1-3.****费用**

      相较于材料及人工，费用应该是最难做到准确分摊的，以费用中主要的折旧及动力为例：

      首先，加工过程中所耗用的设备折旧原则上可以按照资产价值进行划分，但通用资产如车间的厂房所产生的折旧应该如何区分是很难界定的，如果按照各工段资产原值的比重切分，只有在固定资产不发生频繁变动的情况下，才能做到基本准确，如果报告期内有大量的置换资产、新产线及淘汰产线，分拆的工作会非常复杂。

      其次，动力费用如电费、水费更加难以划分到具体产品上，包括前述的通用资产的动力消耗、具体车床的消耗，已经相关消耗有多少是用于生产何种产品的，如果按产品的维度去归集动力消耗，估计财务就疯球了...

      所以，费用的分摊也类似于人工，会制定标准定额，但是这个定额的偏差率一般是要比人工高很多的...但是莫得办法，如果要落到具体产品上，有偏差的分配也比完全不分摊的要好一些....

      总而言之，如果想做好存货最先要搞清楚的就是料工费的分配方式，具备同行业经验的话可以比对同行业类似项目的分配方式；如果是位萌新建议访谈时了解公司的上下游或同行业竞争对手，是否有近期报会的，如果有的话，可以查看其报会招股书，一般会披露生产工艺及成本核算流程；最惨的就是完全没有对标公司...只能参考大的基本原则审核现有核算方式是否科学、合理。摸着良心说，上来就能把成本核算拾到干净的项目少之又少，特别是在和收入确认时点调整混在一起，约等于把申报期的成本重算一遍...

     **1-4.****收发存报表（进销存）**

     收发存报表是按照材料、半成品或成本的唯一编号记录各期期初数量/金额，入库数量/金额，出库数量/金额及结存数量/金额的汇总表，也**是存货最重要的数据来源和审计程序的起点，**做存货不要收发存或成本报表的，一看就知道是萌新...至于这个东东有什么用，后面讲程序的时候会具体说。
