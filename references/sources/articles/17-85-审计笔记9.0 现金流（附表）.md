# 审计笔记9.0 现金流（附表）

**附表的使用用途**  
其实大多数人并不太理解现金流量表附表存在的意义....说实话，我一开始也不理解...主表能编平就求爷爷告奶奶了...附表里还一万种勾稽关系，怼数一时爽，复核火葬场...  
简单举个例子来说明吧，比如同规模的两家公司，净利润均为2000万，经营性活动净现金流流量均为3000万，但是一家的经营性应收的减少为1000万，另一家的经营性应付的增加为1000万。那差别的点在哪里那？

-   第一家（应收变动）经营性应收的减少表示公司应收账款以前年度欠款收回，经营性活动是靠收回款项形成的，是真正意义上的“经营活动流入”；
-   第二家（应付变动）经营性应付的增加表示公司实际的经营性流量表现良好是欠账欠出来的...并不是因为回款良好产生的资金流入。  
    仔细一比对，不难看出第一家整体的情况粗略看下来是要比第二家好，但是如果不披露现金流量表附表，财报使用者很难观察到这一重要信息。

  
（后来有小朋友补充，这里再额外说两句：所谓财务报表分析，主要是基于个人经验及财务数据表现得出的一个大体上的判断，并不代表准确客观结果。原则上来说，经营性应付增加表示公司对供应商有较大的信用期空间，但出来混早晚都是要还的，上面的表述确实有瑕疵，还是需要结合多期报表比对的）  
**下面举几个简单的例子，纯属个人理解，如有误人子弟之处，还望指正。**  
 

**应收账款（不考虑增值税）**

**一、经营性应收款**

在不考虑增值税情况下，确认收入分录（应收账款借方发生额）分录：  
借：应收账款  2500  
   贷：主营业务收入  2500  
在实际取得回款时，确认回款分录（应收账款贷方发生额）分录：  
借：银行存款  1700  
   贷：应收账款  1700

  
上述分录中：在不考虑成本费用等情况下，主营业务收入即净利润增加值为2,500；现金流量表经营活动销售商品收到现金1,700。应收账款资产负债表逻辑：期初余额+借方发生额\-贷方发生额\=期末余额，在不考虑税费情况下，将收入及现金流发生额与借贷方金额进行置换，即：

应收账款期初余额 1000

+主营业务收入（净利润）2500

\-销售商品收到的现金（经营活动净现金流量）1700

\=应收账款期末余额 1800  
推导得出

净利润  
+应收账款期初余额\-应收账款期末余额  
\=经营活动净现金流量

其中应收账款期初余额减期末余额即为公式法所述经营性应收减少

**二、非经营性应收款**  
现金流附表不平...不要只挤经营性应付，有的时候也会有非经营性应收的影响。举个例子：  
比如报表本年新设，无期初各项余额及发生额，本期预付100元为支付购买设备首付款，在编制现金流量表主表时，该笔业务指向为投资活动购建资产。如果按照上面所述公式法计算：

净利润 = 0应收账款期初余额\-应收账款期末余额\=0-100=-100经营性活动净现金流量\= 0 - 100 = -100与实际经营活动现金流量不符  
形成差异的原因：**因为公式法实际计算时，是以经营性应收款为前提，经营性****...****经营性****...****经营性，实际支付的预付款属于投资活动，故在计算****/****填列附表时需予以剔除**，如果不剔除应挤经营性应付就会出现如下情况：

净利润\=0经营性应收本期减少\=-100经营性应付本期增加\=100经营性净现金流量\=0  
看着逻辑没毛病...但实际上报表中没有负债类项目...一般情况下，是不会怼秃的，但如果存在大额预付、拆借资金形成其他应收款，附表内金额就会显的异常突兀..所以如果发现附表内经营性应收、应付项目双向差异巨大，扪心自问是不是怼数了...

**应付账款（不考虑增值税）**

**一、经营性应付款**

在不考虑增值税情况下，成本结转（应付账款贷方发生额）分录：  
借：主营业务成本  2700  
   贷：应付账款  2700  
注：为便于理解，此处简化原材料至产成品分配计算过程

在实际支付采购款项，付款（应付账款借方发生额）分录：  
借：应付账款  1500  
   贷：银行存款  1500  
上述分录中：在不考虑成本费用等情况下，主营业务成本即净利润为\-2,700；现金流量表经营活动购买商品支付现金1,500。应付账款资产负债表逻辑：期初余额+贷方发生额\-借方发生额\=期末余额，在不考虑税费情况下，将收入及现金流发生额与借贷方金额进行置换，即：

应付账款期初余额 1000

+主营业务成本（净利润）2700

\-购买商品支付的现金（经营性活动净现金流量）1500

\=应付账款期末余额2200  
区别于经营性应收，上表中所示净利润为负数，实际经营活动净现金流量为流出1500，所以在推导过程中要按照负数运算，即：  
应付账款期初余额\-净利润+经营性活动净现金流量\=期末余额  
推导得出

净利润  
+应付账款期末余额\-应付账款期初余额  
\=经营活动净现金流量

**二、非经营性应付款**  
非经影响应付款主要指由投资活动或资金拆借性往来款形成的应付结余，与应付未付设备采购款、投资款等，也包括自关联单位取得未偿还资金拆借款项。

**1.****购买资产**

**(1)** **支付非经营活动款项**  
举例来说：假设期初有100元应付款，是上期应付未付购买资产款项，本期实际支付100元，在编制现金流主表时，该笔现金流出会作为购建资产所支付的现金（投资活动）100元，

在实际支付采购款项，付款（应付账款借方发生额）分录：  
借：应付账款  100  
   贷：银行存款  100

  
有且仅有一笔分录时：公司当期净利润为0，经营性现金流量0，投资活动净流出100，应付账款期末较期初减少100元。  
如简单按照公式法填列，则

净利润 0+经营应付本期增加（期末减期初）\-100=经营性现金流量\=-100**与主表实际经营性活动** **0** **不符**  
形成差异是由于本期应付账款变动金额为非经营性应付款项变动，并非公式法所述经营性应付账款变动，故支付投资活动款项导致的应付账款余额变化在使用公式法时，应进行还原：

公式法：应付账款期末余额\-期初余额\=0-100=-100还原项目：支付投资活动款项\=100实际经营性应付变动额\=-100+100=0

**(2)** **新增非经营活动款项**  
该还原的得还原...该剔除的得剔除..  
举例来说：假设期初有0元应付款，本期期末购买固定资产300元，设备款未付，形成期末余额300元。未计提折旧，净利润为0。在编制现金流主表时，无现金流量发生额（钱都没付，哪来的发生额...）

在实际采购固定资产，应付账款贷方分录：  
借：固定资产  300  
   贷：应付账款  300\`  
     
有且仅有一笔分录时：公司当期净利润为0，经营性现金流量0，投资活动净流出100，应付账款期末较期初减少100元。  
如简单按照公式法填列，则

净利润 0+经营应付本期增加（期末减期初）300=经营性现金流量\= 300与主表实际经营性活动 0 不符  
形成差异是由于本期应付账款变动金额为非经营性应付款项变动，并非公式法所述经营性应付账款变动，故因采购固定资产导致的应付账款余额变化在使用公式法时，应进行还原：

公式法：应付账款期末余额\-期初余额\= 300-0 = 300还原项目：购买固定资产未支付款项\=300实际经营性应付变动额\= 300 - 300= 0  
应付账款在实际编制过程中一般是最麻烦的...不管是主表还是附表，主要是由于要将特殊项目进行梳理形成备查，在实际附表逻辑检验的过程中进行还原。  
 

**财务费用**  
附表中财务费用不等于报表财务费用....为啥那？第一、在编制现金流主表时，财务费用\-利息收入在无特殊情况下，一般作为收到其他经营活动项目，实际体现为经营活动净流入，以分录为例子：

在确认财务费用时，记账分录为：  
借：财务费用  -100  
   借：银行存款  100 （收到其他与经营活动相关）\`  
     
有且仅有一笔分录时，公司当期净利润为100，经营性现金流量100  
如果将财务费用发生额直接填列在附表中，则

净利润\=100 （财务费用\-利息收入）财务费用\= -100 （报表金额）经营活动净现金流量\= 0  
与主表逻辑不符（经营性活动净流入100）**现金流量表附表实际填列的财务费用金额为非经营性活动产生的财务费用，即借款利息支出、贴现息、汇兑损益、未确认融资费用摊销值等项目，一般不包括利息收入、手续费等项目**  
第二、实际需填列在附表财务费用中项目如上所述，为非经营性活动，why？以分录为例子：

在确认借款财务费用时，计提及支付记账分录为：  
借：财务费用  200  
   贷：银行存款  50 （分配股利偿付利息所支付的现金）  
   贷：应付利息  150\`  
   
有且仅有一笔分录时:公司当期净利润为\-200，经营性现金流量0，筹资活动现金流出50  
如果将财务费用发生额直接填列在附表中，则

净利润\= -200 （财务费用\-利息收入）财务费用\= 200 （报表金额）经营活动净现金流量\= 0  
为什么单独把应付利息列出来，是因为有时候质控老师会问为什么现金流偿付利息与财务费用附表金额不勾稽..这个就是原因，因为赖着没付...
