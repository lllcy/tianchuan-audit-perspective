第三部分 会计处理的合规性

**一、\*\***主要审核问题\*\*

监管机构针对应收票据会计处理的合规性，主要围绕**《企业会计准则第22\*\***号——\***\*金融工具确认与计量》**和**《企业会计准则第23\*\***号——\***\*金融资产转移》**，提出以下具体问题：

1.   **金融资产的分类与列报（初始确认）**

·     **问题：**发行人对应收票据是如何进行分类的？为何部分在“应收票据”列报，部分在“应收款项融资”列报？

·     **关注点：**

·     分类是否严格遵循了“业务模式”和“合同现金流量特征”测试？

·     将信用等级较高的银行承兑汇票分类为以公允价值计量且其变动计入其他综合收益的金融资产（列报为“应收款项融资”）的依据是否充分？（如**气派科技、呈和科技**的案例）

·     分类标准是否一贯适用？同行业处理是否可比？

2.   **终止确认的合规性（后续计量）**

·     **这是最核心、被问及最多的问题。**

·     **问题：**已背书或贴现的应收票据，是否满足终止确认的条件？

·     **关注点：**

·     **风险与报酬转移判断：**是否对所有已背书/贴现的票据进行了“主要风险和报酬是否转移”的实质性判断？

·     **追索权条款：**贴现或背书协议中是否附有追索权条款？这是判断风险是否转移的关键。（文档中**金冠电气、罗克佳华**等多次提及）

·     **承兑人信用等级：**是否对不同承兑人（如6大行、上市股份行、信用等级一般银行、财务公司、企业）的票据采用了不同的终止确认政策？（如**博力威、宏远股份**的核查程序）

·     **银行承兑汇票 vs.** **商业承兑汇票：**是否区分对待？通常，对信用等级较高的银行承兑汇票，因拒付风险极低，背书/贴现时可终止确认；而商业承兑汇票因信用风险较高，通常不终止确认。（如**容百科技、晟楠科技**的明确论述）

3.   **坏账准备的计提**

·     **问题：**应收票据（尤其是商业承兑汇票）的预期信用损失（坏账准备）计提是否充分、合理？

·     **关注点：**

·     **计提政策：**是否建立了明确的坏账计提政策？商业承兑汇票是否与应收账款一样，按账龄连续计算并计提坏账？（如**金杨股份、冠龙股份**强调“账龄连续计算”）

·     **计提比例：**计提比例是否谨慎？是否与同行业可比公司一致？（如**金冠电气**将“与同行业对比”作为核查程序）

·     **单项计提：**对于已知存在重大风险的出票人/承兑人，是否进行了单项计提？

4.   **特殊业务的会计处理**

·     **问题：**应收账款与应收票据之间的转换、票据贴现息处理、票据池业务等如何处理？

·     **关注点：**

·     **转换处理：**收到客户票据抵偿应收账款时，账龄是否连续计算？（**湖北万润**的问询中涉及）

·     **贴现息处理：**贴现息的计算是否准确，是计入财务费用还是冲减营业收入？会计处理是否正确？

·     **票据池业务：**融入票据池的票据是否应终止确认？相关义务是否确认为负债？

5.   **现金流量表的列报**

·     **问题：**票据的收取、背书、贴现等业务在现金流量表中如何列示？

·     **关注点：**

·     收到的票据款是否作为“销售商品、提供劳务收到的现金”？

·     背书支付采购款是作为“购买商品、接受劳务支付的现金”还是作为经营活动现金流的调整项？

·     贴现取得的现金是作为经营活动还是筹资活动现金流？（**金杨股份**的核查中专门分析了现金流问题）

**二、\*\***核查要求与程序\*\*

中介机构需执行以下极为详尽的核查程序来应对上述问题：

|
**核心问题**

|

**具体核查程序与取证要点**

|

**程序目的与证据形式**

| |
| |

|

**1.** **分类与列报**

|

1. **获取并复核会计政策：**检查发行人金融资产分类的具体会计政策，特别是“业务模式”的书面定义。 2. **抽样测试：**抽取分类为“应收款项融资”的票据，核对承兑人名称，并通过公开信息核查其信用等级是否与政策一致。 3. **同行业对比：**查阅同行业上市公司年报，对比分类政策的一致性。

|

**证据：**书面会计政策、票据清单、信用评级报告、同行业年报截屏。**目的：**验证分类的合理性及一贯性。

|
|

**2.** **终止确认（核心中的核心）**

|

1. **获取清单：**获取报告期各期末“已背书未到期”和“已贴现未到期”的应收票据完整清单。 2. **审阅协议：\*\***逐笔检查**（或大额抽样）贴现协议和背书凭证，**重点关注是否含有追索权条款**。这是最关键的程序。 3. **承兑人分析：**对清单中的承兑人进行分类统计（如6大行、上市股份行、城商行、财务公司、企业），判断发行人对不同类别承兑人的终止确认政策是否恰当。 4. **期后测试：**检查这些已终止确认的票据在期后是否被正常兑付。**如有被追偿案例，则证明终止确认错误**。 5. **重新判断：\*\*根据核查到的追索权条款和承兑人信用情况，重新评估发行人的终止确认处理是否符合《企业会计准则第23号》的规定。

|

**证据：**终止确认票据清单、贴现协议/合同（关键证据）、银行兑付回单、承兑人信用等级分类表。**目的：**获取充分证据以支持或反驳发行人终止确认的会计处理。

|
|

**3.** **坏账计提**

|

1. **复核计提政策：**获取发行人的坏账计提政策，评估其合理性。 2. **重新计算：**对应收票据的坏账准备计提进行**重新计算**，验证计算的准确性。 3. **账龄测试：**对于商业承兑汇票，测试其账龄划分是否准确，是否连续计算。 4. **同行业对比：**将发行人的计提政策、计提比例与同行业可比公司进行对比分析，判断是否存在重大差异。 5. **期后回款测试：**检查期后无法兑付的票据情况，反向验证坏账计提是否充分。

|

**证据：**坏账计提政策、坏账计算表、同行业对比分析表、期后回款统计。**目的：**验证坏账计提的充分性和准确性。

|
|

**4.** **特殊业务处理**

|

1. **追踪转换链条：**针对应收账款转为应收票据的业务，抽查相关原始凭证，确认账龄计算是否连贯。 2. **复核贴现息：**抽取贴现业务，重新计算贴现息，并检查其会计处理科目是否正确。 3. **检查票据池协议：**如有票据池业务，获取相关协议，检查会计处理是否符合“质押融资”或“出售”的实质。

|

**证据：**记账凭证及后附原始单据、贴现息计算表、票据池业务合同。**目的：**验证特殊业务处理的准确性。

|
|

**5.** **现金流列报**

|

1. **勾稽核对：**将票据的收取、背书、贴现等数据与现金流量表相关项目进行勾稽核对。 2. **复核编制底稿：**检查发行人现金流量表的编制底稿，了解其对票据业务的具体列报逻辑，并判断其是否合规。

|

**证据：**现金流量表编制底稿、勾稽核对表。**目的：**验证现金流列报的准确性。

|

三、**结论标准**

经过上述详尽核查后，中介机构需在回复中给出斩钉截铁的结论，典型表述如下（综合自文档）：

·     **“\*\***发行人对应收票据的分类和列报符合《企业会计准则第22\***\*号》的规定。”**

·     **“\*\***发行人已背书或贴现的应收票据（主要为信用等级较高的银行承兑汇票）的主要风险和报酬已经转移，终止确认的会计处理符合《企业会计准则第23\***\*号》的规定。”**（同时需明确说明对商业承兑汇票的处理方式）

·     **“\*\***发行人应收票据坏账准备的计提政策与同行业可比公司不存在重大差异，计提是充分的。”\*\*

·     **“\*\***发行人应收票据相关的会计处理在所有重大方面符合企业会计准则的规定。”\*\*

**总结：**会计处理合规性的核查，本质上是**用会计准则作为尺子，去丈量发行人的每一项票据业务**。核查过程要求**穿透合同条款（特别是追索权）、验证信用等级、复核计算过程、检查账务处理**，最终必须形成清晰的专业判断：发行人的处理是“符合”还是“不符合”准则。任何模糊或回避的结论都无法通过审核。

第四部分 应收票据的流转与勾稽关系

**一、\*\***主要审核问题\*\*

监管机构要求发行人对应收票据的整个生命周期进行清晰的量化还原，并确保财务数据间逻辑严密，无重大矛盾。主要问题可拆解为：

1.   **整体勾稽关系（宏观层面）**

·     **问题：**报告期各期，应收票据的**期初余额、当期收到金额、当期背书金额、当期贴现金额、当期到期承兑收回金额、期末余额**之间的数学关系是否成立？是否存在无法解释的重大差异？

·     **关注点：**这是最基础的勾稽测试，旨在验证账面记录的整体准确性。公式通常为：期末余额 = 期初余额 + 当期收到金额 - 当期背书金额 - 当期贴现金额 - 当期到期承兑金额 ± 其他调整项 

2.   **明细勾稽关系（微观与分类层面）**

·     **问题：**上述勾稽关系是否在**银行承兑汇票**和**商业承兑汇票**分类别、**终止确认**和**未终止确认**分类别下均成立？

·     **关注点：**防止发行人用一类票据的变动去掩盖另一类票据的问题。例如，文档中**金杨股份**的表单就明确区分了“背书总额”与“背书未终止确认金额”。

3.   **“\*\***收到金额”\***\*的真实性与完整性**

·     **问题：** “当期收到金额”是否完全源于销售回款？是否与当期销售收入、应收账款减少额相匹配？

·     **关注点：**防止将非经营性收到的票据（如借款质押票）或虚增的票据计入，夸大经营现金流。

4.   **“\*\***背书/\***\*贴现金额”\*\***的实质与去向\*\*

·     **问题：** “当期背书金额”是否全部用于支付真实的采购货款？背书给供应商的金额是否与当期采购额、应付账款减少额相匹配？

·     **关注点：**防止票据背书给非供应商（如关联方或第三方进行融资套现），掩盖资金真实流向。

5.   **“\*\***期末余额”\***\*的构成与披露**

·     **问题：**期末余额是否清晰区分为：

·     **期末实际持有票据金额**（企业手中真实的票）

·     **已背书或贴现但未终止确认的票据金额**（企业已转手但仍承担风险的票）

·     **关注点：**确保财务报表充分披露了所有票据风险敞口。文档中**金杨股份**的表单精确展示了⑧期末实际结存金额和⑩期末报表原值的勾稽关系。

6.   **异常情况与特殊调整**

·     **问题：**是否存在**票据到期拒付、票据找零、票据退回、票据质押**等特殊业务？这些业务如何在勾稽关系中体现？

·     **关注点：**任何偏离标准流程的事项都可能是风险点，需要有明确的会计处理并能被勾稽关系所解释。例如，**复旦微电**的勾稽表中就单独列示了“本期退回票据金额60.00万元”。

**二、\*\***核查要求与程序\*\*

中介机构需执行以下程序，以验证上述勾稽关系的准确性与真实性：

|
**核查目标**

|

**具体核查程序与工作底稿要求**

|

**程序目的与证据形式**

| |
| |

|

**1.** **验证整体勾稽关系**

|

1. **编制或复核勾稽表：**获取发行人编制的类似**金杨股份**的详细勾稽表（必须分票据类型）。 2. **重新计算：**独立计算  期末余额 = 期初余额 + 当期收到 - 当期背书 - 当期贴现 - 到期收回 ± 其他调整 ，核对是否勾稽相符。 3. **追踪差异：**对于发现的任何差异（如**复旦微电**2019年200万元的差异），必须追查至底层凭证，获取合理解释（如合并范围变动、票据拒付调整等）。

|

**证据：**《应收票据勾稽关系明细表》、差异解释说明及支持性文件。**目的：**从数字上验证总账、明细账的一致性。

|
|

**2.** **验证\*\***“\***\*收到\*\***”\***\*的真实性**

|

1. **来源核对：**从“当期收到金额”中抽样，将票据的**前手背书人**与发行人**客户清单**及销售合同进行匹配。 2. **与收入\*\***/\***\*应收账款勾稽：**将“销售收到的票据金额”与含税销售收入、应收账款贷方发生额进行交叉核对，分析波动是否匹配。

|

**证据：**抽样检查的记账凭证及后附票据复印件、销售合同、客户名单匹配表。**目的：**验证票据来源的经营性和真实性。

|
|

**3.** **验证\*\***“\***\*背书\*\***/\***\*贴现\*\***”\***\*的实质性**

|

1. **去向核对：**从“当期背书金额”中抽样，将票据的**后手被背书人**与发行人**供应商清单**及采购合同进行匹配。 2. **与采购\*\***/\***\*应付账款勾稽：**将“用于支付采购款的票据背书金额”与含税采购额、应付账款借方发生额进行交叉核对。 3. **检查贴现协议：**对贴现业务，检查贴现协议、银行回单，核实贴现金额、贴现息计算的准确性。

|

**证据：**抽样检查的背书凭证及后附票据复印件、采购合同、供应商名单匹配表、贴现协议。**目的：**验证票据支付的经营性和真实性，防止虚假贸易或资金流出。

|
|

**4.** **验证\*\***“\***\*期末余额\*\***”\***\*的准确性**

|

1. **监盘：**对期末持有的**纸质票据进行\*\***100%\***\*监盘**，将票据信息（票号、金额、出票人、承兑人）与财务账面、《应收票据备查簿》记录进行核对。 2. **函证：**对于**已贴现未到期**的票据，向贴现银行发函，确认贴现金额、追索权等条款。 3. **核对备查簿：**将财务账面余额与《应收票据备查簿》的结存记录进行核对，确保账账相符。

|

**证据：**《应收票据监盘表》、银行询证函回函、《应收票据备查簿》。**目的：**验证期末资产的存在性、完整性及权利与义务。

|
|

**5.** **验证特殊业务处理**

|

1. **检查异常记录：**检查账务处理中是否有“票据退回”、“票据找零”、“票据质押”等特殊分录。 2. **追查至业务实质：**对上述特殊分录，追查至原始业务申请、审批记录、对方收据等，判断其商业合理性及会计处理的正确性。

|

**证据：**特殊业务的审批单、协议、银行回单等原始凭证。**目的：**确保所有票据流转活动均被恰当记录和解释。

|

**三、\*\***结论标准\*\*

经过上述详尽核查后，中介机构需在回复中给出量化且明确的结论，典型表述如下（综合自文档）：

·     **“\*\***报告期内，发行人应收票据的期初余额、收到金额、背书金额、贴现金额、到期回收金额、期末金额等各项目之间的勾稽关系相符。”\***\*（金杨股份）**
